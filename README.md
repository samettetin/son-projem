## Quick Start (Angular 15 / Node 18 LTS)
1. **Install Node 18 LTS** (ensure `node -v` ~18.x, `npm -v` >= 9)
2. Install dependencies: `npm install`
3. Run dev server: `npm start` (http://localhost:4200)
